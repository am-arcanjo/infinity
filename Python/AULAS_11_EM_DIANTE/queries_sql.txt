- Criando tabela filmes:

CREATE TABLE filmes (
  id INTEGER PRIMARY KEY AUTOINCREMENT
  , nome VAR(255) NOT NULL
  , diretor VARCHAR(255) NOT NULL
  , ano_lancamento INTEGER NOT NULL
);


- Inserindo filmes:
